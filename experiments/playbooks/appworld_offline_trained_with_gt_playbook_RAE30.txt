[shr-00036] Never create a new song review when one already exists
- Always first retrieve existing reviews using `show_song_reviews(song_id=song_id, user_email=supervisor_profile['email'])` to check the current rating. If a review exists and its rating is not already 1, update it to 1 star using `update_song_review(review_id=song_review_id, rating=1)`. This prevents 409 conflict errors and ensures data consistency with the API's uniqueness constraint (one review per user-song pair).
